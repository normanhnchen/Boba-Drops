# Notes on HTML

## Structuring code

`<!DOCTYPE html>` lets the browser know that the code is using html.

`<html> ... <html>` represents the container for the entire webpage.

`<head> ... <head>` is the behind the scenes; it contains info about the page that the user can't see.

`<title> ... <title>` Changes the text on your browser's tab.

`<body> ... <body>` is the main event: everything that shows up on the screen lives inside here.

## Headings

They are used for titles and subtitles.
They range from `<h1>` (largest in size) to `<h6>` (smallest in size)

## Paragraphs

Used for regular text.
Use the `<p>` tag.

## Links (Anchors)

To link another website, use the `<a>` tag. It also must use the `href` attribute to point to the URL.

## Images

To link an image, use the `<img>` tag. IMPORTANT NOTE: it does not have a closing tag!
